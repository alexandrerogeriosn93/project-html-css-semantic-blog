### Without Live Server

1. Navigate to the project folder on your computer.

### With Live Server (Recommended)

If you are using **Visual Studio Code**, you can use the **Live Server** extension for a better development experience:

1. Install the **Live Server** extension by Ritwick Dey from the VS Code Marketplace.
2. Open the project folder in VS Code.
3. Right-click on `index.html` in the Explorer sidebar.
4. Select **Open with Live Server**.
5. The project will automatically open in your default browser and refresh whenever you save changes.
